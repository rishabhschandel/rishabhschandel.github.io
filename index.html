MDN Web Docs, previously Mozilla Developer Network and formerly Mozilla Developer Center, is the official Mozilla website for development documentation of web standards and Mozilla projects.

Mozilla Developer Network is a resource for developers, maintained by the community of developers and technical writers and hosting many documents on a wide variety of subjects, such as:

  • HTML5
  • JavaScript
  • CSS
  • Web APIs
  • Node.js

For mobile web developers, MDN provides documentation on subjects such as building a HTML5 mobile app, building a mobile add-on, and location-aware apps
